## INSTALLATION

You can download BashTube on any platform by cloning the official Git repository:

```bash
$ pkg install -y git python

$ git clone https://github.com/TrollSkull/BashTube

$ cd BashTube
  
$ pip3 install -r requirements.txt
    
$ python bashtube.py
```
